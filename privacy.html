Privacy Policy

Last Updated: January 16, 2026

Overview

This privacy policy describes how Whoop Personal Analytics ("the Application") handles data accessed through the Whoop API. This is a personal data analysis tool developed and used solely by a single individual for personal health analytics and statistical research purposes.

Key Point: This is a local, personal-use application. Your Whoop data is accessed only by you, for your own analysis, and remains on your local machine.

Data Collection and Use

The Application accesses the following data from your Whoop account via the official Whoop API:

  • Recovery data: Recovery score, heart rate variability (HRV), resting heart rate
  • Cycle data: Daily strain, average heart rate during cycles
  • Sleep data: Sleep duration, quality, start and end times
  • Workout data: Activity type, duration, accumulated strain
  • Profile data: Basic user profile information
  • Body measurements: Height, weight, maximum heart rate

Data Storage and Security

  • All data is stored locally on your personal computer
  • No data is transmitted to external servers or third parties
  • No data is shared, sold, or disclosed to anyone
  • Access tokens are stored securely in local configuration files

Data Retention

Data is retained on your local machine for as long as you choose to keep it. You may delete all data at any time by removing the application files from your computer.

Third-Party Services

This Application uses only the official Whoop API to access your data. No other third-party services are involved in data collection, processing, or storage.

Your Rights

As the sole user of this personal application, you have complete control over your data:

  • You can revoke API access at any time through your Whoop account settings
  • You can delete all locally stored data at any time
  • You control what analysis and processing occurs

Changes to This Policy

This privacy policy may be updated periodically. The "Last Updated" date at the top of this page indicates when the policy was last revised.

Contact

This is a personal project. If you have any questions about this privacy policy, you may contact the developer through the GitHub repository associated with this project.

Compliance

This Application is designed for personal use and complies with:

  • Whoop API Terms of Service
  • OAuth 2.0 security standards
  • General data protection principles
